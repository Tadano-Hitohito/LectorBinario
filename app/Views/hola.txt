Este proyecto implementa una aplicación web en PHP/CodeIgniter que aplica operadores booleanos (AND, OR, XOR, NOT) al procesamiento de archivos ROM de Game Boy Advance/Color. El sistema funciona como un procesador especializado que utiliza lógica digital para analizar la estructura binaria de las ROMs, identificar secuencias de texto mediante técnicas de enmascaramiento, y facilitar su extracción para traducción. Desarrollado con CodeIgniter, la herramienta incluye un editor hexadecimal interactivo en el navegador, visualizador de operaciones booleanas en tiempo real, y algoritmos optimizados para búsqueda de patrones. El proyecto demuestra la aplicación práctica de álgebra booleana en problemas de ingeniería inversa, ofreciendo una solución accesible desde cualquier navegador web.
